Here are my Software Development Projects.

Hostel Management Portal

Hostel Management Portal is a streamlined solution for managing complaints and announcements in hostels and residence halls. Key features include:

  • Complaint Lodging: Students can lodge complaints with statuses like Accepted, Rejected, or Pending.
  • Hierarchical Escalation: Complaints can be escalated to higher-ranking officers for better resolution.
  • Announcement Section: View the latest management updates with role-based visibility.
  • Filters: Filter complaints by hostel, category, year, or date for precise results.
  • Privacy: Public and private complaint visibility options ensure transparency and confidentiality.

Tech Stack: HTML5, Bootstrap, PHP, MySQL.

Project Management System

Project Management System (PMS) is a comprehensive solution to manage team projects efficiently. Key features include:

  • Issue Tracking: Admins assign issues with categories, priorities, and due dates.
  • Role Management: Admins add users; both users and admins can assign tasks.
  • Kanban Board: Visual task tracking to monitor progress.
  • Reports: Generate reports with insights using dynamic charts.
  • Security: JWT tokens and Auth0 for secure authentication.

Tech Stack: AngularJS, Node.js, Express, MongoDB.

Meetup - Real-Time Chat Application

Meetup is a versatile real-time chat application designed for seamless communication. It offers:

  • Real-Time Chat: 1-1 chat and group chat functionality.
  • Video Conferencing: Live video calls with multiple participants.
  • Rich Messaging Features: Emoji reactions, threaded conversations, and notifications.
  • Secure Authentication: Google Auth0 and JWT for secure login.

Tech Stack: React, Chakra UI, Express, MongoDB.
